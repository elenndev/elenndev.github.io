Verify Passcode

 Repositório GitHub

Uma API com dois endpoints para gerar e lidar com a verificação de códigos de acesso de usuários que serão enviados por e-mail e possuem um tempo de expiração, construída com Node.js, Express, MongoDB e Nodemailer.

  • Recebe username e email e gera um código de acesso (passcode).
  • Salva o código de acesso no banco de dados e o envia para o email do usuário via email utilizando o nodemailer.
  • Valida se o código de acesso enviado pelo usuário é compatível com o que está salvo para aquele username/email.
  • Utiliza CORS para permitir requisições do domínio permitido.
