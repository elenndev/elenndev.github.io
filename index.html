Blog: Music Archive

 Reposiório GitHub Website

Blog desenvolvido utilizando React e com um backend desenvolvido em python. Permite ao leitor responsividade e preferência de tema de cor, e ao usuário administrador, segurança e possibilidade de começar a escrever suas publicações e continuar em outro momento, com a funcionalidade dos rascunhos. Para mais detalhes e visualização do dashboard, o template dele foi disponibilizado no github, e pode ser acessado aqui.

  • Sistema de login e proteção de rotas
  • Responsividade
  • Modo dark/light
  • Interação com a API do Spotify
  • Requisições a API que permite a leitura e gerenciamento de publicações, rascunhos e demais informações do blog guardados no banco de dados.
  • Criar e editar rascunhos

Blog Dashboard
Reposiório GitHub

Dashboard de blog desenvolvido utilizando React para gerenciamento de posts e dos demais conteúdos do blog. Na área de dashboard o usuário tem acesso ao controle do conteúdo do blog e tem a possibilidade de ao começar a escrever uma nova publicação, poder salvar apenas como rasunho e terminar em outro momento.

  • Leitura e gerenciamento de publicações
  • Criar, Listar e editar rascunho de publicações.
  • Criação automatizada de iframe spotify embed: Com essa funcionalidade ao passar o link padrão de compartilhamento da playlist, o embed é formado automaticamente.
  • Interação com a API do Spotify para buscar albúns e assim salvar o albúm selecionado.
